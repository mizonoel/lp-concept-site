## 📝 概要
「大会を目指す上級者」から「ダイエット目的の初心者」まで、幅広いターゲットに刺さる熱量のあるデザインを追求。視線誘導を意識したセクション構成により、ユーザーを自然にアクションへ促す導線設計を行いました。

## 🎨 デザインコンセプト：「Dynamic & Energetic」
- **配色**: 黒と赤をベースに「力強さ」を表現。一方で十分な白余白を確保し、情報の視認性と洗練された印象を両立。
- **視覚的インパクト**: 太めのタイポグラフィと大胆なビジュアルを採用し、ページを開いた瞬間にモチベーションを高める演出。
- **UX設計**: 視線誘導を意識したレイアウトにより、サービスの特徴からCTA（行動喚起）までスムーズな体験を提供。
